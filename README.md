# Twin_Matrix: The Ultimate Skill Distiller 🧪

> *"知識の共有から実務の伝授へ。あらゆる組織と達人の叡智を蒸留（Distill）する究極のナレッジマネジメント"*

[English](README_en.md) | [中文](README_zh.md) | [日本語](README.md)

## 概要 (Overview)

**Twin_Matrix**は、特定の達人、チーム、組織の企業文化、あるいは業務マニュアルなどから、「思考（Mind）」「暗黙知（Tacit Knowledge）」「実務プロセス（Execution Process）」を完全に蒸留（Distill）し、新たな自律型AIスキル（`[TargetName]_Twin.md`）として出力するメタ・スキル（Meta-Skill）です。

## 究極のナレッジマネジメントとしてのTwin_Matrix

従来の組織におけるナレッジマネジメントは、ドキュメントの共有やマニュアルの蓄積にとどまっていました。しかし、言語化されていない「社内政治の機微」「なぜこの構成にしたかの裏の意図」「思考の順序（ワークフロー）」といった**暗黙知（Tacit Knowledge）**は、そのままでは決して伝承されませんでした。

Twin_Matrixは、この限界を突破します。対象者のサンプルテキスト（ブログ、チャット、評価レポートなど）をAIに読み込ませることで、以下の3つの要素を完全にリバースエンジニアリングしてカプセル化します。

1. **認知的フレームワーク (Mental Models & Anti-patterns)**
   彼らが「何を重視し、何を絶対にやらないか」という意思決定の土台。
2. **暗黙知の言語化 (Context & Tacit Knowledge)**
   字面には書かれていない、力関係や配慮などの「行間」。
3. **実行プロセスの完全複製 (Workflow Reverse-Engineering)**
   「どのような情報をどう集め、どう論理立ててアウトプットするか」というタスク解法のステップ化。

### 知識の「共有」から、実務の「伝授」へ
Twin_Matrixが生成した「Twinファイル」を他のメンバー（あるいはAIエージェント）に読み込ませるだけで、そのナレッジは即座に実務へと伝授されます。
「どのように考えるか」だけでなく、「どのように手を動かし、どういう成果物を出すか」までを完全にインストールできる、これが**究極のナレッジマネジメント（Ultimate Knowledge Management）**の形です。

## 使い方 (Usage)

AIアシスタント（ClaudeやGPT、Antigravityなど）に対し、`Twin_Matrix.md` のプロンプトと一緒に、蒸留したいターゲットのサンプルデータを渡すだけです。

**入力例:**
> 「Twin_Matrixを使って、この添付した『〇〇チームの業務マニュアルとSlackのチャット履歴』を蒸留して。独自の暗黙知と実行プロセスを継承する新しいTwinスキルファイルを生成して」

結果として、対象チームの完全なコピーとして自律動作するMarkdownスキルファイルが出力されます。
